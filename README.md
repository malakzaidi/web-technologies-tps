# Web Technologies Training Repository

## Overview
This repository is dedicated to learning and practicing **web technologies**, including **HTML**, **CSS**, **JavaScript**, **Bootstrap**, and other related tools and frameworks. It serves as a collection of exercises, projects, and examples to help you master the fundamentals and advanced concepts of web development.

Whether you're a beginner or an experienced developer, this repository provides a structured way to practice and improve your skills in building modern, responsive, and interactive web applications.

## Introduction
This repository is designed to provide hands-on training in web development. It includes a variety of exercises and projects that cover the core technologies used in building websites and web applications. Each exercise is accompanied by detailed instructions, code examples, and solutions to help you understand and apply the concepts effectively.

---

## Technologies Covered
The repository focuses on the following technologies:

- **HTML**: Structuring web content.
- **CSS**: Styling and designing web pages.
- **JavaScript**: Adding interactivity and dynamic behavior.
- **Bootstrap**: Building responsive and mobile-first websites.
- **Other Tools**: Introduction to tools like Git, VS Code, and more.

## How to Use This Repository
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/malakzaidi/web-technologies-tps.git
   ```
2. **Navigate to the Exercise or Project**:
   Go to the specific folder for the technology or topic you want to practice.
3. **Follow the Instructions**:
   Each exercise or project includes a `README.md` file with instructions and explanations.
4. **Experiment and Modify**:
   Feel free to modify the code, experiment with new ideas, and test your understanding.
5. **Check Solutions**:
   Solutions are provided for most exercises. Compare your work with the provided solutions to learn best practices.

## Exercises and Projects
### HTML Exercises
- **Basic HTML Structure**: Learn how to create a well-structured HTML document.
- **Forms and Inputs**: Practice creating forms with various input types.
- **Semantic HTML**: Use semantic tags to improve accessibility and SEO.

### CSS Exercises
- **Basic Styling**: Apply styles to HTML elements using CSS.
- **Flexbox and Grid**: Master layout techniques with Flexbox and CSS Grid.
- **Animations and Transitions**: Create engaging animations and transitions.

### JavaScript Exercises
- **Variables and Functions**: Understand the basics of JavaScript syntax.
- **DOM Manipulation**: Learn how to interact with the Document Object Model.
- **Event Handling**: Add interactivity to your web pages with event listeners.

### Bootstrap Exercises
- **Responsive Layouts**: Build responsive designs using Bootstrap's grid system.
- **Bootstrap Components**: Use pre-built components like navbars, cards, and modals.
- **Customizing Bootstrap**: Learn how to customize Bootstrap themes and styles.

### Projects
- **Personal Portfolio**: Build a responsive portfolio website using HTML, CSS, and JavaScript.
- **E-commerce Landing Page**: Create a landing page for an online store using Bootstrap.
- **Interactive Quiz App**: Develop a quiz application with JavaScript.
